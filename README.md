# ♟️ Chess Game

A Chess Game developed in Python.

## Features

- ♟️ Classic Chess Board
- 🎮 Two Player Mode
- 🖱️ Mouse Controls
- ⚡ Fast Gameplay
- 📊 Game Statistics

## Requirements

- Python 3.x
- Pygame

## Installation

```bash
git clone https://github.com/tushargawande-hub/chess-game.git
cd chess-game
pip install pygame
python chess.py
```
